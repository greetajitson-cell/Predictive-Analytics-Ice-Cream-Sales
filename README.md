# Predictive Analytics Using Historical Ice Cream Sales Data

## Objective
Forecast future ice cream sales using historical data.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Visualization
4. Model Building
5. Sales Forecasting

## Outcome
Successfully predicted future sales trends using Linear Regression.
